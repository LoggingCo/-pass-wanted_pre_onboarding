# 원티드 프리온보딩 백엔드 - 선발 과제

원티드 백엔드 프리온보딩 선발과제 레퍼지토리

## ERD

users - 사용자 테이블
emposts - 채용공고 테이블
emapplys - 채용지원 테이블 (중간 테이블 적용시 관계 이상으로 관계 테이블로 승격)
companys - 회사 테이블
